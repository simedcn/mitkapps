Step 2 : Segmentation


Now that you successfully data, let's segment the different parts of the organs. But first...

1. Select the data to segment

It can happen that DICOM files contain multiple pixel 3D datasets. However, the segmentation occurs only from a single pixel 3D dataset. You must indicate to OrganPrint which dataset to use in order to segment the organ parts.
Go on the right panel and select the data node that you want to segment

2. Create a segmentation node

Click on the button "New segmentation" on the left panel.
